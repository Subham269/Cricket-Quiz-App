# 🏏 Cricket Quiz App

A timed cricket trivia quiz built with vanilla HTML, CSS and JavaScript.

🔗 **[Live Demo](https://cricketquizappsubham.netlify.app/)**

---

## About

A 10-question cricket quiz that tests your knowledge of cricket history, records and legendary moments. Each question has a 30-second timer — answer before time runs out or the quiz moves on automatically.

---

## Features

- 10 hand-picked cricket trivia questions
- 30 second countdown timer per question
- Instant visual feedback — green for correct, red for wrong
- Score tracked throughout the quiz
- Auto-advances when timer runs out
- Restart button to play again
